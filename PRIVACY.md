# Privacy Policy - Alicia AI

**Data Storage:** Your Groq API key and all chat history are stored locally on your device using Chrome's storage API. No data is sent to Alicia AI servers.

**Third-Party Sharing:** Chat content is only sent to Groq's API (groq.com) for AI processing. Your API key controls this communication. We do not collect, sell, or share your data.

**User Control:** You can clear all stored data by uninstalling the extension or using the Clear Chat button in the interface.

**Last Updated:** June 2026
